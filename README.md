# loop-entry
## Java
```java
String[] a = {"AB-", "CD-", "EF"};
for( int i = 0; i < a.length; i++ ) {
    System.out.println(a[i]);
}
for( String value: a ) {
    System.out.println(value);
}
```
## C#
```cs
string[] a = {"AB-", "CD-", "EF"};
for( int i = 0; i < a.Length; i++ ) {
    Console.WriteLine(a[i]);
}
```
## PHP
```php
$a = ["AB-", "CD-", "EF"];
for( $i = 0; $i < count($i); $i++ ) {
    print "{$a[$i]}\n";
}
foreach ( $a as $value  ) {
    print "{$value}\n";
}
```

## JavaScript
```javascript
var a = ["AB-", "CD-", "EF"];

    console.log( a[i] );
}
```
